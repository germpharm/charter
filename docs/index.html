Charter

AI governance layer. Local-first. Open source. Three layers: hard constraints, gradient decisions, self-audit.

$ pip install charter-governance

Layer A: Hard Constraints

Things your AI must never do. No exceptions. You define the rules. The system enforces them.

Layer B: Gradient Decisions

Actions that require human judgment above certain thresholds. Financial transactions, external communications, sensitive data access.

Layer C: Self-Audit

The system reviews what it did and reports honestly. Accountability, not perfection.

Quick Start

# Create a governance config
$ charter init
 
# Generate enforceable AI instructions
$ charter generate
 
# Run a governance audit
$ charter audit
 
# See your identity
$ charter identity
 
# Check everything at a glance
$ charter status

Features

Domain Templates

Healthcare (HIPAA), Finance (SOX), Education (FERPA), General. Governance presets for regulated industries.

Pseudonymous Identity

SHA-256 hash chain. Every action signed. When you verify your identity, all prior work transfers to you.

Context Isolation

Separate work and personal knowledge. No cross-context flow by default. Governed bridging with both-party consent.

Network Layer

Register expertise, connect data sources, recognize who shaped you. Local-first, shareable when ready.

Works With Any AI

Generate a CLAUDE.md, a system prompt, or raw YAML. Charter governs the AI, not the other way around.

Kill Triggers

Conditions that immediately stop all AI activity. Ethics declining? Audit bypassed? The system shuts itself down.

Philosophy

The value of AI is not in the tokens. Tokens are going to zero. The value is in the humans who provide judgment, context, and ethics.

Charter is the governance layer that makes human judgment enforceable on AI systems.

Open source because we don't need more rent seekers. We need human creativity to thrive while being accountable for what it creates.
